Solución:

Los pasos que ocurren al momento de hacerle click a un enlace son:

  1. El navegador verifica el caché para buscar la IP del enlace
  2. Si no lo encuentra en el cache, solicita informacion al dns
  3. Se inicia una conección TCP con el servidor
  4. El cliente envía un paquete SYN al servidor preguntando si está abierto a nuevas conexiones
  5. Si el servidor está disponible responde con un SYN/ACK paquete
  6. El cliente reconoce el SYN/ACK y envía un ACK
  7. La conexión TCP es establecida
  8. El navegador envía una petición HTTP get
  9. El servidor reconoce la petición
  10. El servidor envía la respuesta
  11. El cliente recibe la respuesta desde el servidor
